[ cassette - portfolio ] [ project - gallery study ]


isometric view

this project was a demonstration of developing a piece which you can walk around (i.e. something that looks good from the inside and the outside), themed around an art gallery or museum. this room of the museum is showcasing an exoplanet, from a faraway galaxy.


this project features complex post-processing, some interesting shading techniques, a bit of graphic design for the posters, and a bit of a PSX style overall, using Eevee for rendering.
